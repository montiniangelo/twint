Analisi

Requisiti funzionali

Analisi

  • Gestione marker con info dettagliate (titolo, descrizione, contatti, link) e ricerca per categorie / filtri.

Architettura tecnica

Tecnico

  • Servizi esterni: Google Maps JavaScript API per rendering mappa, marker, geocoding/places se necessari.

Tecnico
